
Настройка базы данных
  mysql> CREATE DATABASE phones CHARACTER SET utf8 COLLATE utf8_unicode_ci;

Настройка пользователя (в версии MySQL 8.0)
  mysql> CREATE USER 'phuser'@'localhost' IDENTIFIED WITH mysql_native_password BY 'phpass';

Назначение привилегий
  mysql> GRANT ALL ON `phones`.* TO 'phuser'@'localhost';
  mysql> flush privileges;

Загрузка дампа данных
  > mysql -u root -p phones < install/phonebook.sql

Известные проблемы:
-------------------

- есть ошибка в коде: если при поиске по тексту вывод занимает более одной
  страницы, то последующие страницы не формируются - неверная ссылка перехода.
  Для решения необходимо либо изменить вид ссылки на 
           /?query=ив&page=1 
  (вместо кривой: /?query=ивpage/1/), либо сменить блок пагинатора (навигации
  по страницам).

- при окончании редактирования должности не переносит в соответсвующее
  подразделение (проверить шаблон на наличие переменных "podr"->"unit" )
